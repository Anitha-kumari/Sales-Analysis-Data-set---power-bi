📊 Sales Analysis Dashboard | Power BI
📖 Project Overview
The Sales Analysis Dashboard is an interactive Power BI report designed to monitor and analyze sales performance across different regions, products, salespersons, and payment methods. The dashboard provides real-time business insights using dynamic filters, KPI cards, and interactive visualizations to support data-driven decision-making.
🎯 Project Objectives
Monitor overall sales performance.
Analyze profit and profit margin.
Track total orders and average order value.
Compare sales by region and product.
Analyze sales trends over time.
Evaluate payment method performance.
Enable interactive filtering for detailed analysis.
📌 Key Performance Indicators (KPIs)
💰 Total Sales
💵 Total Profit
📈 Profit Margin (%)
📦 Total Orders
🛒 Average Order Value (AOV)
📊 Dashboard Features
KPI Cards (Sales, Profit, Profit Margin, Orders, AOV)
Monthly Sales & Profit Trend
Region-wise Sales Analysis
Product-wise Sales Performance
Payment Method Distribution
Salesperson Performance
Interactive Filters (Region, Product, Payment Method, Salesperson, Store Location)
Date Range Slicer
